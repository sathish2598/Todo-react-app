# Todo React App

A minimal todo application built with React and Vite. Tasks are stored in your browser (localStorage).

## Features

- **Add** tasks with the input and Add button
- **Complete** tasks by checking the checkbox
- **Edit** tasks by double-clicking the text
- **Delete** tasks with the × button
- **Persistence** — your list is saved automatically

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 5
- CSS (no UI library)
